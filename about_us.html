About Us - Temperature Control Project

Temperature Control System using ESP32

This project is a smart temperature control system built with an ESP32 microcontroller. It uses a DS18B20 digital temperature sensor to continuously monitor the environment.

Based on the measured temperature, a servo motor is controlled dynamically, and a relay is triggered to manage external devices (like heaters or fans). The system uses a PID (Proportional-Integral-Derivative) algorithm to maintain the temperature near a desired setpoint.

All data — including live temperature, setpoint, and PID parameters (Kp, Ki, Kd) — are synced in real-time to Google Firebase, and users can adjust these parameters remotely through a custom Android application built using MIT App Inventor or a web interface.

Project Features

  • Real-time temperature monitoring
  • Dynamic PID parameter adjustment
  • Cloud connectivity with Firebase
  • Relay control based on PID output
  • Servo motor angle mapping based on temperature
  • Mobile and web app control
